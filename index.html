Projects

Kidlat: Lightning-Fast EV Charging Finders

Co-Developers: Alisha Chua, Javier Del Rosario, Niccolo Jimenez, Stefano Santos and Philipp Suarez

Project Timeline: May 2024 - July 2024

Technology Stack: Flutter, Figma & Github

Kidlat is an MVP mobile app developed for the ITISDEV course to address the challenge of locating electric vehicle (EV) charging stations in the Philippines. The app provides a solution through interactive maps, real-time charger availability, pricing information, and user reviews, helping EV users, businesses, and travelers find and access chargers easily. Built with Flutter, Kidlat delivers a seamless user experience with smooth navigation and real-time data. The app was prototyped in Figma and managed via GitHub, offering additional features like route planning and notifications to enhance EV journeys.

Ticketnet

Co-Designers: Alisha Chua and Nina Sorongon

Project Timeline: June 2024 - July 2024

Technology Stack: Figma

TicketNet, an online ticketing platform in the Philippines, faced challenges with its cumbersome checkout process and difficulties in keeping users updated on announcements, promotions, and event details. To address these issues, we redesigned the user interface of the website to streamline the ticket purchasing process, making it quicker and more efficient. Additionally, we improved the accessibility of updates and information, and ensured the site is fully optimized for mobile devices. This redesign enhances user experience by providing a more intuitive and responsive platform for purchasing tickets and staying informed about events.

CoFeed

Co-Developers: Miguel Caparos, Azeliah Martinez and Keisha Villanueva

Project Timeline: January 2024 - April 2024

Technology Stack: Figma, JavaScript, Node.JS, CSS, Github

Cofeed brings together coffee, community, and collaboration to create the ideal platform for students and remote workers. Tackling the challenge of finding comfortable, productive spaces, Cofeed lets users browse coffee shops and study spots, write reviews, and connect with like-minded individuals. The name combines "coffee" and "feed," reflecting its focus on coffee culture and shared experiences. More than just a search tool, Cofeed helps users discover hidden gems, build connections, and strike the perfect balance between work and relaxation.

Lashlux Studio Database

Co-Developers: Miguel Caparos, Azeliah Martinez and Keisha Villanueva

Project Timeline: October 2023 - November 2023

Technology Stack: SQL, JavaScript

The LashLux Studio database application addresses the challenge of managing an eyelash beauty salon's growing operations. The system simplifies tracking services, customer and employee information, and appointment scheduling, while also managing a loyalty program. It provides an efficient solution by automating service updates, customer preferences, appointment management, and employee performance tracking. Additionally, it calculates customer loyalty points, streamlines reward redemption, and tracks metrics like customer attendance and revenue, ensuring smoother salon operations and enhanced customer experience.

Seoul Bites Vending Machine

Co-Developers: Keisha Villanueva

Project Timeline: May 2023 - August 2023

Technology Stack: Java, Github

The Seoul Bites Vending Machine program simulates a vending machine factory, providing users with the ability to create and test vending machines. The program offers a variety of functions, such as adding items, replenishing money, printing transactions, checking inventory, purchasing items, and more, to enhance user interaction. The graphical user interface (GUI) is designed to be simple and straightforward, ensuring easy navigation and a user-friendly experience.
